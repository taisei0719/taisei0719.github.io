BestShift

Automatic Shift Management System

プロジェクト概要

Overview

アルバイトスタッフのシフト提出から、管理者のシフト表作成までをワンストップで自動化・効率化するWebアプリケーションです。 現在は開発の初期フェーズとして、バックエンドの基盤構築およびデータベース設計を中心に進めています。

開発の背景

Motivation

多くの店舗で行われている「手書きやSNSでのシフト回収」「表計算ソフトへの手入力」というアナログな工程を自動化し、 ヒューマンエラーの削減と管理コストの大幅な短縮を目指しています。

技術スタック・開発環境

Tech Stack

バックエンド

  • Python / Flask
  • Flask-SQLAlchemy (ORM)
  • SQLite (開発用DB)

フロントエンド

  • TypeScript
  • HTML5 / CSS3

現在の進捗

Current Status

  • データベース設計: Shop, User, Shiftモデルの実装とリレーションの定義が完了。
  • 初期化スクリプト: 店舗コードの自動生成およびテストユーザーの自動登録機能を実装。
  • UI設計: モバイルファーストなシフト提出画面のプロトタイプを作成中。

今後の展望

Future Work

今後は、独自のアルゴリズムを用いた「希望シフトに基づく自動割当機能」の実装を予定しています。 また、バックエンドをPostgreSQLへ移行し、スケーラビリティを確保したマルチテナント対応を進める計画です。
